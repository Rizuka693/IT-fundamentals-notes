## 🖥️ Linux Basics / Linux 基本

Some basic commands I learned:  
学習した基本コマンド：

- `pwd` → show current directory / 現在のディレクトリを表示  
- `ls` → list files and directories / ファイルとディレクトリの一覧表示  
- `cd` → change directory / ディレクトリの移動  
- `mkdir` → create new folder / 新しいフォルダの作成  
- `rm` → remove files / ファイルの削除  
- `cp` → copy files / ファイルのコピー  
- `mv` → move/rename files / ファイルの移動・名前変更

## 🌐 Networking Basics / ネットワーク基礎

Basic concepts:  
基本的な概念：

- **IP Address** → unique address for a device / ネットワーク上の機器を識別する番号  
- **DNS** → translates domain names to IP / ドメイン名をIPアドレスに変換する仕組み  
- **Gateway** → connects network to internet / ネットワークとインターネットをつなぐ役割  

Basic commands / 基本コマンド：
- `ping google.com` → check connection / 接続確認  
- `ifconfig` / `ip a` → check IP / IPアドレスの確認  

---

## 🛠️ Troubleshooting / トラブルシューティング

Basic steps:  
基本的な対応手順：

1. Check physical connection / ケーブルやWi-Fiの確認  
2. Restart device / 再起動  
3. Check IP address / IPアドレス確認  
4. Test with `ping` / `ping`で確認  
5. Check DNS settings / DNS設定確認  

Example case / 対応例：

**Problem:** Cannot access internet  
**問題:** インターネットに接続できない  

**Cause:** DNS issue  
**原因:** DNS設定の問題  

**Solution:** Change DNS to 8.8.8.8  
**対応:** DNSを8.8.8.8に変更
